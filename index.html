Welcome to unofficial Dead by Daylight API documentation.

With this API you can't get informations for Killer, survivors; Perks and more. You can use query in URL if you search for a specific response.

JUMP TO
GET Perks GET Killers GET Survivors GET Currencies

Changing language

You can change language with the query lang
French and English is actually available so you can use 

English = API_URL + "?lang=en" 
 French = API_URL + "?lang=fr"



Images

All images is hosted on my github
You can simply use link directly in the API, or you can use perk_tag and name_tag to build image link.
